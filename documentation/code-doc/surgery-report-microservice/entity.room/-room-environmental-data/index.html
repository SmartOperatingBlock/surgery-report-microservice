RoomEnvironmentalData

@Serializable
data class RoomEnvironmentalData(val temperature: Temperature? = null, val humidity: Humidity? = null, val luminosity: Luminosity? = null, val presence: Presence? = null)

Wraps all the environmental data associated to a Room. So it describe:

  • the temperature inside the room

  • the humidity inside the room

  • the luminosity inside the room

  • the presence of someone in the room All the data may be not present.
